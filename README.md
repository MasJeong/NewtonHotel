# NewtonHotel Team Project

- Java, JSP/Servlet, AJAX, HTML, CSS, JavaScript, Jquery, Bootstrap 등을 사용하여 제작한 호텔 예약 웹 사이트입니다.
- MVC 패턴과 ActionFactory 패턴을 사용하였습니다.
- 사용 API:
  - 도로명 주소 팝업 API
  - 카카오 로그인 API
  - 네이버 로그인 API
